# forspace-react

Новостной блог о мире технологий.
В стадии разработки.

### Технологии и стек:
- React 17.0.2
- i18next
- Jest
- Loki
- Storybook
- Feature-Sliced Design (архитектурная методология)

#### Для установки приложения:

```
npm i - установка зависимостей
npm run start - запуск проекта
npm run build - сборка проекта
```
